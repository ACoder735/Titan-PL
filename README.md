# The **Titan** Programming Language
The new Titan programming language(PL), based on Python, and mixes the worlds of the System and Scripting PLs.
It is built on top of the exellent Go (Golang) programming language, and is designed to be fast, efficient, and easy to use.

## Some Titan code examples:
```titan
typeln("Hello, World!")
x := 5
if x == 5{
  typeln("x is equal to 5")
}
```

```titan
for i:=1 to 10{
  typeln(Str(i)) // Titan does not alow you to print raw numbers
}
```
## Why Titan?
Titan is designed to be the **PERFECT** proggramming language, choosing to be:

**Simple**

**Safe**

**Fast**

**Boilerplate-free**

It is also designed to be a great language for both beginners and experienced programmers, with a simple syntax and powerful features.
